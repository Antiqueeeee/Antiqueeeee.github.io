打包、分发、部署

打包:就是把你软件运行所需的依赖、第三方库、敏件打包到一起，变成一个安装包
分发:你可以把你打包好的”安装包”上传到一个镜像仓库，莫他人可以非常方便的获取和安装
部署:拿着”安装包”就可以一个命令运行起来你的应用，自动模拟出一摸一样的运行环境，不管是在Windows/Mac/Linux。

镜像、容器

镜像:可以理解为软件安装包，可以方便的进行传摇和安装。
容器:软件安装后的状态，每个软件运行环境都是独立的、隔离的，称之为容器。

安装过程

安装后报错了这个错误：Docker Desktop requires a newer WSL kernel version.
需要更新wsl更新包，下载地址是：https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi
安装好了等几分钟就能运行docker了

使用Docker快速安装软件

docker run

一键拉取镜像

docker compose

在包含docker-compose.yml文件的路径下，按照yml文件中的要求安装容器。
